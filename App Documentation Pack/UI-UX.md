# Neo — UI / UX Specification

# Design Philosophy

Neo should feel:
- premium
- calm
- technical
- operational

Inspired by:
- Codex
- Cursor
- Linear
- Raycast

Avoid:
- gimmicky AI aesthetics
- excessive glassmorphism
- neon cyberpunk visuals

---
