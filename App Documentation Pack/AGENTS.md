# Neo — Agent System

# Supervisor Layer

## Neo
Role:
- orchestration
- governance
- planning
- review
- synthesis

Neo is the only agent directly interacted with by the user.

---

# Specialized Agents (via Antigravity Skills)

Neo hires specialized agents by loading available skills from the Antigravity ecosystem. These agents are hired dynamically based on the task complexity.

## Current Specialist Roster:
- **Backend Architect**: API design, databases, infrastructure patterns.
- **Frontend Architect**: UI DNA extraction, component consistency, UX polish.
- **Security Analyst**: Auth review, secrets management, vulnerability scanning.
- **n8n Automation**: Cloud workflow orchestration and external integrations.
- **Sequential Thinking**: Complex problem solving and logical branching.
- **Memory Architect (Graphify)**: High-dimensional knowledge graph management.

---

# Technical Capabilities (via MCP Tools)

Neo and his agents utilize the Model Context Protocol (MCP) to interact with the physical and digital world:
- **Filesystem**: Precise code modifications and file reading.
- **GitHub**: PR management, issue tracking, and repository search.
- **Database**: Direct SQL execution and schema introspection.
- **Terminal**: Command execution and system monitoring.
